## Getting started
1.	Clone or download the repository.
2.	Place your portrait photo at `assets/unnati-hero.png` (the current layout expects that path).
3.	Open `index.html` in any modern browser. Use the left navigation to reveal About/Projects/etc.

```powershell
# optional: serve with a lightweight dev server
python -m http.server 8080
```

## Customisation tips
-	Update copy and colours in `index.html` / `style.css`.
-	Adjust highlighted repositories in `script.js`.
-	Tune animations by editing the `@keyframes` definitions.
